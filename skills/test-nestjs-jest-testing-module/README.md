# test-nestjs-jest-testing-module

Write unit tests for NestJS applications using the Testing Module and Jest.

## What it does

Provides guidance for writing unit tests of NestJS providers, services, and controllers in
isolation using `Test.createTestingModule` and Jest.

> Status: this skill is a work in progress — the detailed instructions in `SKILL.md` are still
> being authored.

## When to use

- Writing isolated unit tests for NestJS services, providers, or controllers with Jest

For full request/response (API integration) tests, use `test-nestjs-supertest` instead.

## Installation

Install only this skill:

```bash
npx skills add dariopalminio/agile-sddf-extension --skill test-nestjs-jest-testing-module
```

Or install all skills in the repository:

```bash
npx skills add dariopalminio/agile-sddf-extension --all
```

After installation the skill is invoked automatically by context or directly by name.
